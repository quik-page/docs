插件打包

node build.js

运行这个命令，在out/目录就会输出打包结果。

index.js
version.txt //这是随你定义的update生成

index.js就是插件本体，也就是可以使用“从第三方文件添加插件”安装的，本地安装的插件不支持更新。

你也可以将这两个文件上传到你的网站，然后就可以使用“从第三方链接添加插件”安装，但一定要配置好跨域，在线安装的插件就可以更新。当你有新版本时，只需要将新的index.js,version.txt替换原来的就可以了。
