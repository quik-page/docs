主题

相当一部分插件都是提供另一个风格的主题样式，QUIK起始页内置了一个简单的API。

像这样添加一个主题选项，当用户在设置选择了这个主题后，body就会添加mytheme这个className :

quik.custom.addTheme('mytheme','我的主题')

然后就可以像这样自定义样式：

body.mytheme ...{
    /*...*/
}

主题颜色可以通过以下方式获取：

quik.custom.getColor() // dark | light
